A fully responsive Expense Tracker built with the **MERN stack**, featuring JWT authentication, income and expense management with category tracking, interactive data visualizations, and Excel export functionality.

`MongoDB` `Express.js` `React` `Node.js` `JWT`

---

### Coachmate — Fitness Mobile Application
A cross-platform workout tracker built with **React Native and TypeScript**, featuring real-time workout tracking, AI-powered exercise guidance via **Groq**, a content-managed exercise library using **Sanity CMS**, and secure authentication with Clerk and Google OAuth.

`React Native` `TypeScript` `Expo` `Groq` `Sanity CMS` `Clerk`

`Java` `Spring Boot` `Angular`
